# static-webpack

## フロント開発

### ディレクトリ構造

```txt
├── public  # ドキュメントルート
└── src     # コンパイル前ソース
```

### 推奨nodeバーション
```
node.js(version: v16.13.1)
```

### 依存ファイルインストール
```
npm install
```

### 開発開始
```
npm run start
```

### 開発終了
```
npm run build
```
